# Sales & Customer Behavior Analytics

## Project Overview


Team analytics project focused on exploring customer purchasing behavior, sales performance, and engagement patterns in an e-commerce environment.

The project combines:

data cleaning and exploratory data analysis (EDA) in Python (Google Colab)
feature engineering and customer segmentation
behavioral and revenue analysis
interactive business intelligence dashboard development in Tableau

Key analysis areas:

sales trends and category performance
customer segmentation and lifetime value
engagement vs spending behavior
demographic and device-based analysis
geographic revenue distribution

Tools & Technologies:

Python (Pandas, NumPy, Matplotlib, Seaborn)
Google Colab
Tableau
CSV datasets
